# Gungnir
Gungnir: Java一些常用的工具型的设计

## Result

* **Result**: 执行结果

* **Failure**: 错误信息

* **PojoResult**: 保存单个实体的Result

* **CollectionResult**: 保存集合的Result

* **PageResult**: 带有分页信息的集合Result

* **OffsetResult**: 带有偏移信息的集合Result

* **MapResult**: 保存键值对的Result

* **PairResult**: 保存一对值的Result

* **TripleResult**: 保存三个值的Result
